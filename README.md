# University-Project TEST DATA FOR PERSONAL LEARNING
Python project that generates PDF analytics reports from university admissions data, including enrollment trends, demographics, scholarships, and academic performance visualizations.

#Generate Chart and it shows:
1.International students enrolled each year (2021–2025)
2.Department intake trend year by year (line per dept)
3.Online vs Offline applications shift over years
4.English language levels of students per year
5.Male vs Female students each year
6.Top 10 EU countries sending students
7.Scholarship rate % per year
8.Average GPA by department and year (colour heatmap table)
